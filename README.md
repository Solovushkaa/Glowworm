# Glowworm

Glowworm is a cross-platform desktop application for exchanging data over the network. Built with Qt and QML, it provides a modern graphical interface for file sharing, synchronization, and remote storage management.

✨ Features:
- Network data exchange – seamless file transfer between users over the network
- WebDAV support – connect to WebDAV servers for remote file access and synchronization
- Modern QML interface – responsive and visually appealing user experience
- Relay server integration – enables communication even in complex network environments
- User authentication – secure login with credential management
- Modular architecture – organized into managers, network layer, and utility modules

🛠️ Technology Stack:
- Language: C++20
- Framework: Qt (Core, Network, QML)
- UI: QML / Qt Quick
- Build System: CMake
- Protocols: WebDAV, custom network protocols
- Platforms: Windows, Linux, macOS (cross-platform)
