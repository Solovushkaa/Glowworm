# Glowworm

Glowworm is a cross-platform desktop application for network data exchange

---

## Interface

There are areas on the main screen:

 - 1: A window for displaying files and directories of the file system
 - 2: A window for displaying files and directories
of the connected user's file system
 - 3: A window for displaying active downloads

![image](https://github.com/user-attachments/assets/bb814c14-5974-49f3-b91b-a8de0fb65c22)

The main screen offers the user several basic actions:
 - 1: "Open menu" – open a menu with additional settings
and connections
 - 2: "Share key" – open a window with the connection key
 - 3: "Connect" – connect to the selected user
 - 4: "Download" – get the file from the user
 - 5: "Open access" – grant access to connect to
the user's device
 - 6: "Select a file system item" – select
a file system item to download, delete, rename it
 - 7: "Continue/Stop downloading" – stop or continue
the download

![image](https://github.com/user-attachments/assets/5caca2bf-e90c-47f1-8c74-376ac7507906)

There are also
some additional elements on the main page:
 - 1: The name of the file system
 - 2: Active directory
 - 3: Connection indicator
 - 4: Download data
